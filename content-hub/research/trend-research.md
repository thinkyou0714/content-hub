# トレンド調査手順書(X / Grok)

流行の把握と「日本人の心の動かし方」の学習は、**X(タイムライン)と Grok** を一次ソースにする。
Web検索は事実確認用の二次ソースとして使い分ける。調査結果は [idea-backlog.md](idea-backlog.md) に記録して生成の材料にする。

## なぜ X / Grok なのか

- 流行の半減期が最も短い場所がX。Zennやブログに現れる頃には一巡している
- Grok はXの投稿をリアルタイムに検索・要約できるため、「今なにがバズっていて、どんな言い回しが刺さっているか」を最速で取れる
- バズった日本語投稿の**構造**(冒頭フック、改行位置、感情の起伏、数字の使い方)は、日本人読者の心の動かし方の生きた教材になる

## 調査の頻度と所要時間(人間タスク)

| 頻度 | 作業 | 目安時間 |
|---|---|---|
| 毎日(朝) | Xタイムライン巡回。刺さった投稿を3件ブックマーク | 10分 |
| 週1(週初) | Grokに下記プロンプトを投げ、結果をネタ帳に貼る | 20分 |
| 月1 | ネタ帳の棚卸し。柱([strategy/content-pillars.md](../strategy/content-pillars.md))との対応を見直す | 30分 |

## Grok に投げるプロンプト(コピペ用)

### 1. 週次トレンドスキャン

```text
ここ7日間の日本語のXで、「AI活用」「副業」「個人開発」「自動化(n8n・Claude Code等)」に
関連して大きく拡散した投稿を10件挙げてください。
各投稿について以下を教えてください:
1. 投稿の要旨(1行)
2. 拡散した理由の分析(内容・タイミング・書き方のどれが効いたか)
3. 冒頭1行目の書き方の特徴
```

### 2. 心を動かす構造の分析

```text
最近の日本語のXで、技術・副業系のジャンルで「共感」を大量に集めた投稿を5件挙げて、
それぞれの文章構造を分解してください:
- 冒頭フックの型(問いかけ/実績数字/失敗告白/あるある など)
- 感情の流れ(どこで読者の感情が動くか)
- 締めの型(余韻/行動喚起/自己開示 など)
共通パターンがあれば抽出してください。
```

### 3. ネタの需要検証

```text
「(ネタ候補のテーマ)」について、日本語のXでの直近1ヶ月の反応を調べてください:
1. このテーマはどのくらい話題になっているか
2. どんな切り口の投稿が反応を得ているか
3. まだ語られていない切り口(スキマ)はどこか
4. このテーマで発信する場合の注意点(批判されやすいポイント)
```

### 4. 飽和チェック(記事化の前に必ず)

```text
「(記事タイトル案)」というテーマの記事を書こうと思います。
同種のコンテンツが日本語圏(X・Zenn・note)でどの程度出回っているか、
代表例を挙げつつ「飽和度」を5段階で評価してください。
差別化するなら何を足すべきかも提案してください。
```

## 調査結果の記録ルール

1. Grokの回答は鵜呑みにしない。**投稿の実在と数字は必ずX上で目視確認**(ハルシネーション対策)
2. ネタ帳([idea-backlog.md](idea-backlog.md))には「テーマ/出典/柱/反応の根拠/鮮度(日付)」をセットで書く
3. 他人の投稿は**構造を学ぶ**。文言のコピーや軽微な改変での流用は禁止(規約・信用・著作権すべての面でNG)
4. 鮮度が30日を超えたトレンド系ネタは自動的に価値が落ちたとみなし、月次棚卸しで整理する

## AIエージェントへの委任範囲

- Grok回答の整形・ネタ帳への転記・柱タグ付け → AIに委任可
- Web検索での事実の裏取り(公式ドキュメント・一次情報の確認) → AIに委任可
- **X上での目視確認と「これを書くかどうか」の判断 → 人間**([workflows/human-tasks.md](../workflows/human-tasks.md))
