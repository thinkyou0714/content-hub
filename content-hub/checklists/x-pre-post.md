# チェックリスト: X投稿前

承認フロー(n8n → 人間)の最終目視。1投稿あたり1〜3分で回す前提の軽量チェック。

## 毎投稿(30秒)

- [ ] 1行目だけ読んで続きが気になるか
- [ ] 数字・固有名詞に創作・記憶違いがないか
- [ ] 煽り表現(「誰でも」「必ず」「稼げる」)が入っていないか
- [ ] 外部リンクが本文に入っていないか(リプ欄方式)
- [ ] 「自分の声」として違和感がないか(AIっぽければ承認せず差し戻す)

## 該当時のみ

- [ ] 収益・実績に触れる投稿: 期間・コスト・前提を併記したか
- [ ] 他者への言及: 手法・ツールへの言及に留まり、人格・アカウント批判になっていないか
- [ ] 案件・紹介絡み: PR表示が必要な関係性ではないか(ステマ規制)
- [ ] 画像添付: 図解の誤字・機密(APIキー・個人情報)の写り込みがないか
- [ ] トレンド便乗: 元ネタを目視で確認したか(Grok出力の鵜呑み禁止)

## 差し戻しの基準

以下は修正ではなく生成からやり直す(プロンプト改善のシグナルとして記録)。

1. 型([../templates/x-post-patterns.md](../templates/x-post-patterns.md))から外れた構成
2. 実体験の捏造・数字の水増し
3. 同じ型・同じ言い回しの連続(量産感はスパム判定・信頼低下の両リスク)

差し戻しが週2回を超えたら、プロンプト([../prompts/x-single-post.md](../prompts/x-single-post.md))自体を修正してコミットする。
