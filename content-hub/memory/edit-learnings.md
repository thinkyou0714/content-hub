# 編集学習メモリ(edit-learnings)

AI初稿への人間の修正から抽出した「次回から先回りするためのルール」。
**note・Zenn・Xいずれの生成でも、このファイルを必読コンテキストに含めること。**

運用([README.md](README.md) 参照):

- `npm run note:learn -- <slug>` が「自動抽出(未確認)」の末尾に追記する
- 月次レビューで人間が確認し、有効なものを「確定ルール」へ移す(外れは削除)
- 確定ルールが増えたら style-guide やプロンプト本体へ昇格させる
