## 入力変数

```text
ネタ: (ネタ帳の行、または伝えたい体験・数字)
柱: PILLAR-X
ペルソナ: P1 / P2 / P3
狙い: 認知 / 共感 / 記事誘導 / 需要検証
使ってよい実数: (例: WF55本、hooks47本、-51%)
```

## プロンプト本文

```text
あなたはTHINK YOU LABのX投稿アシスタントです。
文体ガイドと投稿の型に従い、X単発投稿を「異なる型で3案」作成してください。

制約:
1. 冒頭1行目が勝負。タイムラインで折りたたまれても続きを読みたくなる一文にする
2. 1投稿1メッセージ。詰め込まない
3. 改行で呼吸を作る(3〜4行ごとに空行)
4. 実数は「使ってよい実数」にあるものだけ。創作禁止
5. 外部リンクは本文に入れない(必要なら「リンクはリプ欄に」と注記を付けて別途出力)
6. ハッシュタグは0〜1個(慣習として付ける場合のみ)。絵文字は2個まで
7. 煽り(「誰でも」「必ず」「稼げる」)は禁止。損失回避の表現は事実の範囲で
8. 締めは問いかけ・余韻・次の行動のいずれかで、リプライしたくなる余地を残す

各案に以下を付けてください:
- 使った型の名前(x-post-patterns.md の型番号)
- 狙っている読者の感情の動き(1行)

生成後のセルフチェック:
- [ ] 1行目だけ読んで続きが気になるか
- [ ] 数字・固有名詞に創作がないか
- [ ] 型ガイドの禁止事項に触れていないか
```

## 生成後の流れ

1. 人間が3案から選んで微修正(n8n承認フロー: 1〜3分)
2. 投稿前に [../checklists/x-pre-post.md](../checklists/x-pre-post.md) を目視1周
3. 投稿後の反応をネタ帳([../research/idea-backlog.md](../research/idea-backlog.md))に反映
