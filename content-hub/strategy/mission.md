# ミッション・ポジショニング

## ミッション

**「AI副業と個人開発のすばらしさを、動いている実物と実数で伝える」** — これがすべての発信の起点。

夢を売るのではなく、再現手順を売る。
「稼げる」と煽るのではなく、「作れる・動く・続く」を見せる。
読者が読み終わったとき「自分にもやれそうだ」と手が動き出す発信をする。

## 発信者の一貫したポジション

| 要素 | 定義 |
|---|---|
| 名義 | THINK YOU LAB |
| 一言ポジション | 自宅ラボで副業自動化システムを実際に動かしている実践者 |
| 実物 | n8n(セルフホスト・WF数十本) + Claude Code + Obsidian + Supabase |
| 差別化 | ロマンを語る記事ではなく「動かし続けた記録」。失敗・コスト・罠を隠さない |
| 立場 | 教祖ではなく実験ノートの公開者。「俺はこうだった」以上の断定をしない |

## 発信で守る3原則

1. **実物主義**: 公開する数字・スクショ・構成図は、実際に動いているものだけ。盛らない・借りない・作らない
2. **再現性主義**: 「すごい」で終わらせず、読者が同じことをできる粒度(コード・設定・手順)まで書く
3. **リスク明示主義**: 収益の話をするときは、かかったコスト・時間・失敗も必ず併記する(景表法・ステマ規制の観点でも必須)

## やらないこと

- 「誰でも」「必ず」「不労所得」などの断定・煽り表現
- 実体験のない手法の紹介(引用する場合は出典明示で「未検証」と書く)
- 他者の下げ(比較は手法・ツールに限定し、人格・アカウントには向けない)
- 情報商材的なクローズド誘導のみの発信(オープンな技術情報が主、有料は深掘りと時短の対価)
